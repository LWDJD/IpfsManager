# 欢迎使用 IpfsManager

这是一个多功能IPFS文件管理器，旨在提供高效、便捷的文件管理体验。

## 当前功能

| 功能项             | 描述                         |
|-------------------|-----------------------------|
| 📂 一键固定文件     | 将文件一键固定在 Crust         |
| 🔄 自动重试         | 自动重试失败的请求           |
| 💻 JAVA17开发       | 使用 JAVA17 进行开发          |

## 未来功能规划

| 功能项             | 描述                         |
|-------------------|-----------------------------|
| 🔎 自动获取CID      | 自动获取 IPFS 中的 CID        |
| 🖥️ 多线程请求      | 多线程发送固定请求            |
| 🌐 HTTP代理        | 提供 HTTP 代理功能            |
| 📁 自动加载索引     | 自动加载文件(夹)索引         |
| 🎫 CID靓号获取      | 获取靓号 CID                 |
| 🎭 选择性开启GUI     | 根据需要选择性开启图形界面   |
| 🎭 选择性添加引导节点  | 根据需要选择性自动添加引导节点   |
